# Mahima Akther - Personal Portfolio

A modern, responsive personal portfolio website for a Computer Science undergraduate student.

## Features

- Clean, professional design
- Fully responsive (mobile & desktop)
- Hero section with name and tagline
- About Me section
- Projects showcase with cards
- Skills section
- Contact information with clickable links
- Smooth scrolling navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript (minimal)
- Font Awesome icons

## Deployment

This project is ready for GitHub Pages deployment. Simply upload the files to your GitHub repository and enable GitHub Pages in the repository settings.
